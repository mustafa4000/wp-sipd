Apa itu WP SIPD?

WP-SIPD adalah aplikasi pendukung dari Sistem Informasi Pemerintah Daerah (SIPD) ,
berbentuk plugin wordpress yang dikembangkan secara opensource di repository github

https://github.com/agusnurwanto/wp-sipd

Data dari SIPD akan dibackup dan dioptimasi menggunakan aplikasi wp-sipd, sehingga
pemda memiliki backup data dan juga bisa mengintegrasikan datanya dengan aplikasi
yang lain seperti SIMDA dan FMIS.
